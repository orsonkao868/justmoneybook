# 就是記帳 App 規格表

> 版本：v2.0
> 更新日期：2026-04-04
> 網址：https://orsonkao868.github.io/justmoneybook/

## 核心理念

**懶人記帳**：打開就能記，不需要任何設定，兩步驟完成記帳。

操作流程：輸入金額 → 選類別 → 完成（最快3秒）

## 二、記帳功能

### 新增記錄
- 點右下角 ＋ 開啟記帳介面
- 支援支出和收入切換
- 內建數字鍵盤（支援小數點）
- 支援台幣（NT$）和日幣（¥）
- 備註欄位（選填）
- 選擇類別和細項

### 記錄詳細頁面（v2.0 新增）
- 點擊任意記錄開啟詳細頁面
- 顯示完整資訊：類別、細項、金額、貨幣、日期時間、備註
- 可以編輯所有欄位
- 可以刪除記錄

### 記錄列表
- 依日期分組顯示
- 可篩選：全部、支出、收入
- 顯示每日支出小計
- 左右滑動切換日期（v2.0 新增）

## 八、技術規格

### 存檔
- 使用 localStorage
- Key：`moneybook_v1`
- 每筆記錄約 200-300 bytes
- 5MB 空間可存約 15,000-25,000 筆記錄
- 每天記 10 筆可用 4-7 年

### 記錄資料格式
```json
{
  "id": "1712345678901",
  "date": "2026-04-04T11:51:00.000Z",
  "type": "expense",
  "currency": "NTD",
  "amount": 85,
  "cat": "food",
  "subcat": "lunch",
  "note": "便當"
}
```

### PWA 設定
- manifest.json 已設定
- 可加入 iPhone/Android 主畫面
- 全螢幕顯示（無瀏覽器網址列）
- App 圖示：小豬錢包（粉紅色）
- 主題色：#FF6B9D

### 支援平台
- iPhone：Safari → 加入主畫面
- Android：Chrome → 加入主畫面
- 電腦：任何瀏覽器
